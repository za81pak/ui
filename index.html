What's an accordion?

An accordion is a UI component that allows users to expand and collapse sections of content, typically to display information in a compact manner.

When and how should it be used?

It should used when users only need a few key concepts of description of the content on a single page.

What happens if the user clicks on a collapsed card while another card is open?

When a user clicks on a collapsed card (header) while another card is open in an accordion interface, the typical behavior is that the currently open card will collapse (close), and the newly clicked card will expand (open). This creates a clean and organized user experience, ensuring that only one section is visible at a time.
